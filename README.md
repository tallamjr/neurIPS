# NeurIPS

Neural Information Processing Systems Latex template [2021]

#### `latexdiff`

Using `latexdiff`:

```bash
$ git latexdiff HEAD~1 --main src/<paper-name>.tex
```

#### ArXiV

If submitting to arxiv.org, run script in `sbin/make-arxiv-tar.sh` but ensure to replace filenames
appropriately.
